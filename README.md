## Gameplay

- **Auto-run** - Your lobster hero runs automatically
- **Tap/Space** - Jump AND shoot! If already airborne, just shoot
- **Double jump** - Tap again in mid-air for extra height
- **Combo system** - Chain kills for bonus points
- **Endless action** - How far can you go?
